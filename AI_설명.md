# 인공지능(AI) 설명

**AI(Artificial Intelligence, 인공지능)** 는 인간의 사고 능력을 컴퓨터로 구현한 기술입니다.

- **머신러닝**: 데이터를 통해 스스로 학습하는 알고리즘
- **딥러닝**: 인간 뇌의 신경망 구조를 모방한 머신러닝의 한 분야
- **LLM**: GPT, Claude처럼 언어를 이해하고 생성하는 대형 언어 모델
- **지도 학습**: 정답 데이터로 학습 (예: 스팸 메일 분류)
- **비지도 학습**: 정답 없이 패턴을 스스로 찾아 학습
- **강화 학습**: 보상과 벌점을 통해 최적 행동 학습 (예: 게임 AI)
- **활용 분야**: 의료 진단, 자율주행, 번역, 이미지 생성, 코딩 보조 등
- **한계**: 환각(없는 정보를 사실처럼 생성), 편향, 설명 불가능성
- **AI 윤리**: 투명성, 공정성, 안전성, 프라이버시 보호가 핵심 원칙
- **핵심 메시지**: AI는 도구이며, 어떻게 활용하느냐는 사람의 몫입니다.
